# 📊 Investment Portfolio Tracker Dashboard

## Overview
The **Investment Portfolio Tracker Dashboard** is an interactive Power BI dashboard designed to help investors monitor, analyze, and optimize their investment portfolios. It consolidates investment data into meaningful visualizations, enabling users to track portfolio performance, asset allocation, returns, and risk metrics for informed financial decision-making.

## Features
- 📈 Real-time portfolio performance tracking
- 💰 Total investment vs. current market value analysis
- 📊 Asset allocation by investment category
- 📉 Profit & Loss analysis
- 🎯 Return on Investment (ROI) calculation
- 📅 Investment trends over time
- ⚠️ Risk analysis and portfolio diversification insights
- 🔍 Interactive filters for year, quarter, month, investor, and asset type
- 📌 Executive summary dashboard for quick decision-making

## Dashboard Insights
The dashboard provides insights into:
- Total Investment
- Current Portfolio Value
- Overall Profit/Loss
- ROI (%)
- Asset-wise Performance
- Sector-wise Investment Distribution
- Top Performing Assets
- Least Performing Assets
- Monthly & Yearly Investment Trends
- Investor Portfolio Comparison

## Dataset
The dashboard is built using a financial investment dataset containing:

| Column | Description |
|---------|-------------|
| Investor ID | Unique identifier for each investor |
| Investor Name | Name of the investor |
| Purchase Date | Date of investment |
| Asset ID | Unique asset identifier |
| Asset Name | Name of the investment asset |
| Asset Type | Stocks, Mutual Funds, Gold, Bonds, ETFs, etc. |
| Sector | Industry/Sector of investment |
| Units | Quantity purchased |
| Purchase Price | Buying price per unit |
| Investment Amount | Total invested amount |
| Current Market Price | Latest market price |
| Rate of Return (%) | Percentage return |
| Dividend Received | Annual dividend earned |
